# Password Manager 🔐

A full-stack password manager application built with React (Vite) on the frontend and Node.js/Express on the backend.

## 🚀 Features
- Secure password storage
- Password encryption on backend
- Password generator
- User profile management
- Dashboard overview
- Settings and theme support

## 🧱 Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- MySQL (Database)
- Redis (Caching)
- WordPress (CMS demo)
- phpMyAdmin (DB management)
- Nginx (Reverse proxy)
- Docker Compose (Container orchestration)

## 📁 Project Structure
- `/src` - Frontend React app
- `/server` - Backend API
- `/wordpress-docker-lab` - Docker setup (optional environment setup)
